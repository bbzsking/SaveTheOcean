## 一、背景
2023年8月24日，日本开始向太平洋排放核污水，计划持续排放30年，预计排放130万吨。
日本第一批排放计划：向福岛（辐岛）附近的太平洋，17天内排放7800吨核废水！

清华大学团队模拟结果显示，核废水在排放后240天就会到达我国沿岸海域，1200天后将到达北美沿岸并覆盖几乎整个北太平洋，核污水将很快的从太平洋蔓延到全世界！
这将造成严重的生态灾难！
